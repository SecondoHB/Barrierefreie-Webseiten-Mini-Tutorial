Mobile First CSS

Mobile First bedeutet: Du entwickelst zuerst für kleine Bildschirme auf Smartphones, Handys und erweiterst dann für größere Geräte wie Tablets und Desktop.

Warum ist das wichtig?

  • Die meisten Nutzer sind mobil unterwegs
  • Deine Seite bleibt einfacher und schneller
  • Du vermeidest unnötig komplexes CSS

Dein aktueller Ansatz

@media (max-width: 768px) {
  .grid {
    grid-template-columns: 1fr;
  }
}

So denkst du richtig (Mobile First)

/* MOBILE (Standard) */
.grid {
  grid-template-columns: 1fr;
}

/* DESKTOP */
@media (min-width: 768px) {
  .grid {
    grid-template-columns: repeat(3, 1fr);
  }
}

Deine Aufgabe

  • Stelle dein CSS auf Mobile First um
  • Baue eine Box, die auf Mobile untereinander steht
  • Erweitere sie auf Desktop zu 2–3 Spalten

Lösung

Lösung anzeigen 
.box {
  display: grid;
  grid-template-columns: 1fr;
}

@media (min-width: 768px) {
  .box {
    grid-template-columns: repeat(3, 1fr);
  }
}

Checkliste

  • Mobile Layout funktioniert zuerst
  • Desktop wird erweitert
  • Media Queries sinnvoll eingesetzt
