Barrierefreiheit Basics

Barrierefreiheit bedeutet, dass wirklich alle Menschen deine Website nutzen können – unabhängig von Einschränkungen oder Geräten.

1. Warum Barrierefreiheit wichtig ist

  • Nutzer mit Screenreader können Inhalte verstehen
  • Tastatur-Nutzer können alles bedienen
  • Inhalte bleiben auch bei schlechter Sicht lesbar

2. Alt-Texte richtig schreiben

Bilder brauchen eine Beschreibung für Screenreader:

<img src="hund.jpg" alt="Ein Hund läuft entspannt an der Leine in Wien">

schlecht: "hund123.jpg" oder "Bild"

gut: Das Bild beschreibende & sinnvolle Texte

3. Fokus sichtbar machen

Nutzer mit Tastatur brauchen klare Orientierung:

a:focus-visible {
  outline: 3px solid #2b6cff;
  outline-offset: 3px;
}

4. Kontrast ist entscheidend

  • Hellgrau auf Weiß = schlecht
  • Dunkelblau auf Weiß = gut
  • Deine CTA Buttons sind bereits gut lesbar

5. Screenreader denken

Screenreader lesen HTML linear vor. Deshalb ist Struktur wichtig: 

<h1>Haupttitel</h1>
<h2>Untertitel</h2>
<h3>Details</h3>

Keine Sprünge in der Reihenfolge!

Checkliste

  • Bilder haben sinnvolle Alt-Texte
  • Fokus ist sichtbar
  • Kontrast ist ausreichend
  • Struktur ist logisch (H1 → H2 → H3)
