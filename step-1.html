Semantisches HTML

In diesem ersten Schritt lernst du, wie du eine Webseite logisch und barrierefrei strukturierst.

Warum ist das wichtig?

  • Screenreader verstehen deine Seite besser
  • Suchmaschinen können Inhalte einordnen
  • Dein Code wird sauberer und wartbarer

Beispiel

<header>
  <nav>Navigation</nav>
</header>

<main>
  <h1>Haupttitel</h1>
  <p>Inhalt</p>
</main>

<footer>Footer</footer>

Deine Aufgabe

Baue eine kleine Seite nur mit semantischen HTML-Elementen:

  • <header>
  • <nav>
  • <main>
  • <footer>

Lösung

anzeigen 
<header>
  <nav>
    <a href="#">Start</a>
  </nav>
</header>

<main>
  <h1>Meine erste Seite</h1>
</main>

<footer>
  <p>© 2026</p>
</footer>
